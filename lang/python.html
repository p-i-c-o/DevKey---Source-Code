Introduction

Python is a high function, utility based software. In this section of DevKey, you'll learn Python to enter the world of computer coding.

Installation

Python is usually installed on MacOS by default, this is also the case with major linux distributions, like Ubuntu and Fedora. You will have to install it if you want to use it on Windows though.

For the remaining of this guide, we will be using a terminal to run Python, so if you are on Windows (which may be the majority), you will have to run your code through an IDE.

Why is it popular?

Python has become an extremely popular language due to its readability, and simple functions. It is also widely used in back-end web development and game development. The mass availability also gives it its charm.

Your first line of code!

Ready to code in Python for the first time? Here we go! Let's create a piece of code that'll say “Hello World!” back to you when you run it.

print(“Hello World!”)

Let's see what that returns...

Hello, World!

Congratulations, you coded your first line of Python code!



Little mention:

When we test a tiny piece of code, that is only one command or function, we might use this command to simulate running a Python script in the shell (command prompt).

python3 -c '[code]'


That does it for the introduction, now you understand what Python does, and how to start your journey!
