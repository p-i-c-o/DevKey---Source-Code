Introduction

Bash is the basic language for unix based operating systems, such as MacOS, Linux and many more. It's used by millions to control and manage computers that help the internet work.

The Lingo

  • Filesystem : A group of directories that hold files and more subdirectories.
  • Directory : Folder
  • Subdirectory : A folder in a folder
  • Shell : The black box with green text you see in movies, it's used to type in commands
  • Command : A piece of text that commands the computer to do something.

Bash is, simply put, an interface to your computer, it is used to navigate file systems, run commands, control and monitor things. Let's learn how to navigate.


In this section of DevKey, we will use the following filesystem in the majority of our guides.
